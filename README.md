# CoreX storage - s3

This package provides an implementation of the CoreX storage interface for s3.

## Installation
~~~bash
pip install corex-storage-s3
~~~

## Usage
~~~python
from corex_storage_s3.handler import S3Handler
handler = S3Handler()
handler.example_method()
~~~
